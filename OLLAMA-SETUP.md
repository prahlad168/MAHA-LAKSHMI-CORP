## 📋 Setup Ollama

### 1. Install
```bash
curl -fsSL https://ollama.com/install.sh | sh
```

### 2. Download Model
```bash
ollama pull llama3.2
```

### 3. Start & Run
```bash
ollama serve
python gauranga-ollama.py
```
