## 📋 PILIHAN NOTIFIKASI

### 1. ⭐ TELEGRAM (REKOMENDASI UTAMA)

**Kelebihan:**
- Gratis
- Notifikasi langsung ke HP
- Mudah dibuat Bot
- Bisa kirim text, foto, file

**Cara Setup:**

```
Step 1: Install Telegram di HP
Step 2: Buka Telegram, cari @BotFather
Step 3: Ketik /newbot
Step 4: Beri nama bot (contoh: "GAURANGA Reports")
Step 5: Dapat BOT TOKEN (contoh: 123456789:ABCdef...)
Step 6: Buka chat dengan bot baru, kirim "/start"
Step 7: Dapatkan Chat ID dari @userinfobot
Step 8: Simpan TOKEN dan CHAT ID
```

**API Endpoint:**
```
https://api.telegram.org/bot{TOKEN}/sendMessage?chat_id={CHAT_ID}&text={MESSAGE}
```

**Contoh Curl:**
```bash
curl -X POST "https://api.telegram.org/bot123456789:ABC/sendMessage" \
  -d "chat_id=123456789" \
  -d "text=📊 Daily Report: Revenue Rp 5M today!"
```

### 2. 📧 EMAIL

**Kelebihan:**
- Universal (semua HP bisa)
- Tidak perlu install app baru
- Bisa auto-forward

**Setup:**
- Gunakan SMTP server (Gmail, SendGrid, Mailgun)
- Kirim email ke alamat CEO
- Notifikasi email muncul di HP

**Contoh SMTP curl (Gmail):**
```bash
curl -X POST "https://api.sendgrid.com/v3/mail/send" \
  -H "Authorization: Bearer $SENDGRID_API_KEY" \
  -d '{
    "personalizations": [{"to": [{"email": "ceo@company.com"}]}],
    "from": {"email": "reports@company.com"},
    "subject": "Daily Report",
    "content": [{"type": "text/plain", "value": "Report content here"}]
  }'
```

## 🔧 IMPLEMENTATION CODE

### Telegram Bot Script (Python):
```python
import requests

def send_telegram_message(token, chat_id, message):
    url = f"https://api.telegram.org/bot{token}/sendMessage"
    data = {
        "chat_id": chat_id,
        "text": message,
        "parse_mode": "HTML"
    }
    response = requests.post(url, data=data)
    return response.json()

# Usage
TOKEN = "YOUR_BOT_TOKEN"
CHAT_ID = "123456789"
MESSAGE = "📊 Daily Report untuk [CEO_ALIAS_REDACTED]!"
send_telegram_message(TOKEN, CHAT_ID, MESSAGE)
```

### Bash Script:
```bash
#!/bin/bash
TOKEN="YOUR_BOT_TOKEN"
CHAT_ID="123456789"
MESSAGE="$1"

curl -s -X POST "https://api.telegram.org/bot${TOKEN}/sendMessage" \
  -d "chat_id=${CHAT_ID}" \
  -d "text=${MESSAGE}" \
  -d "parse_mode=HTML"
```
