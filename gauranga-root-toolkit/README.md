## 🚀 CARA JALANKAN

### Di Komputer/Server:

```bash
# Clone repository
git clone https://github.com/prahlad168/MAHA-LAKSHMI-CORP.git
cd MAHA-LAKSHMI-CORP/gauranga-root-toolkit

# Jalankan GAURANGA
python gauranga.py
```

### Di Termux HP Android:

```bash
# Install dependencies
pkg update && pkg upgrade -y
pkg install -y python git

# Download
git clone https://github.com/prahlad168/MAHA-LAKSHMI-CORP.git
cd MAHA-LAKSHMI-CORP/gauranga-root-toolkit

# Jalankan

```
