## 🔧 REPORT GENERATOR

```python
def generate_report(report_type: str, agent_id: str, period: dict) -> str:
    """
    Generates standardized report based on type and period.
    """
    templates = {
        'daily': daily_report_template,
        'weekly': weekly_report_template,
        'monthly': monthly_report_template,
        'quarterly': quarterly_report_template
    }
    
    template = templates.get(report_type, daily_report_template)
    
    # Gather data
    data = {
        'agent_id': agent_id,
        'period': period,
        'metrics': get_metrics(agent_id, period),
        'tasks': get_tasks(agent_id, period),
        'achievements': get_achievements(agent_id, period),
        'issues': get_issues(agent_id, period)
    }
    
    return template.format(**data)
```
