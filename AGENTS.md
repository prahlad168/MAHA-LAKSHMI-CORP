## 📥 INPUT & OUTPUT

### Simple Input (yang dibutuhkan untuk generate agent baru):

```yaml
Agent_Input:
  name: "SEO Agent"                    # Nama Agent
  department: "marketing"               # Departemen
  tasks:                               # Tugas utama
      - "keyword_research"
      - "content_optimization"
  targets:                             # Target
      - monthly_leads: 100
  kpis:                               # KPI
      - "keyword_rankings"
  security_level: "internal"          # Level keamanan
```

```markdown
name: "SEO Agent"
department: "marketing"
tasks: ["keyword_research", "content_optimization", "link_building"]
targets: {monthly_leads: 100, organic_traffic: "+50%"}
kpis: ["keyword_rankings", "domain_authority", "backlinks"]
security_level: "internal"
```

### Generate HR Agent:
```markdown
name: "HR Agent"
department: "hr"
tasks: ["recruitment", "onboarding", "performance_review"]
targets: {hire_time: "< 30 days", retention_rate: "> 90%"}
kpis: ["time_to_hire", "retention", "satisfaction_score"]
security_level: "confidential"
```
