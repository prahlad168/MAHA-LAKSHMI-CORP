## 🚀 CARA PAKAI

### Setup di Termux (HP Android):
```bash
# 1. Install dependencies
pkg update && pkg upgrade -y
pkg install -y python git curl wget

# 2. Install Ollama
curl -fsSL https://ollama.com/install.sh | sh

# 3. Download model
ollama pull llama3.2

# 4. Install gTTS (Text-to-Speech)
pip install gtts

# 5. Download GAURANGA
wget -O gauranga_voice_smart.py "https://raw.githubusercontent.com/prahlad168/MAHA-LAKSHMI-CORP/main/gauranga_voice_smart.py"

# 6. Start Ollama
ollama serve

# 7. Run GAURANGA (di terminal baru)
python gauranga_voice_smart.py
```

### Setup di Komputer:
