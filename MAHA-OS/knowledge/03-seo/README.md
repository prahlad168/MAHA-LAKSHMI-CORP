## 🎯 SEO FUNDAMENTALS

### On-Page SEO
```
1. Title tags (50-60 characters)
2. Meta descriptions (150-160 characters)
3. Header hierarchy (H1, H2, H3)
4. URL structure (clean, keyword-rich)
5. Internal linking
6. Image alt text
7. Content optimization
```

### Off-Page SEO
```
1. Backlink building
2. Social signals
3. Brand mentions
4. Guest posting
5. Local citations
```

### Technical SEO
```
1. Site speed
2. Mobile-friendliness
3. XML sitemap
4. Robots.txt
5. HTTPS
6. Structured data
7. Core Web Vitals
```

## 📝 CONTENT SEO

### Article Structure
```
1. H1: Main keyword
2. Intro: Hook + overview
3. H2: Subtopic 1
   - H3: Detail
   - H3: Detail
4. H2: Subtopic 2
5. Conclusion + CTA
```

### Word Count Targets
| Content Type | Words |
|-------------|-------|
| Quick tip | 500-800 |
| How-to guide | 1000-1500 |
| Ultimate guide | 2500-4000 |
| Comparison | 1500-2500 |
