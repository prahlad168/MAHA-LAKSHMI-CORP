## 1. PURPOSE

```
Standar proses untuk client meeting dari preparation hingga follow-up.
```

## 4. BEFORE MEETING

### 4.1 Schedule Meeting

**Steps:**
1. Check calendar availability
2. Send calendar invite
3. Include agenda in invite
4. Send reminder 24 hours before
5. Send reminder 1 hour before

**Calendar Invite Template:**
```
Subject: Meeting with [Client Name] - [Topic]

Date: [Date]
Time: [Time] WIB
Duration: [X] minutes
Platform: [Zoom/Google Meet/WhatsApp/Office]

Agenda:
1. [Topic 1]
2. [Topic 2]
3. [Topic 3]

Meeting Link: [Link]

Please confirm your attendance.
```

---

### 4.2 Prepare Materials

**Checklist:**
- [ ] Review client history in CRM
- [ ] Prepare relevant documents
- [ ] Create presentation if needed
- [ ] Test technology (Zoom, etc.)
- [ ] Prepare notes/questions

**Questions to Prepare:**

**Discovery Call:**
```
1. Tell me about your business?
2. What challenges are you facing?
3. What goals do you want to achieve?
4. What have you tried before?
5. What is your budget?
6. What is your timeline?
7. Who are the decision makers?
```

**Project Check-in:**
```
1. How is the project going?
2. Any concerns or feedback?
3. Are we on track with timeline?
4. Any scope changes needed?
5. Next milestones?
```

## 5. DURING MEETING

### 5.1 Opening (5 minutes)

```
1. Welcome and thank for time
2. Confirm agenda
3. Set expectations (duration, outcome)
4. Introduction if first meeting
```

**Opening Script:**
```
"Halo [Name], terima kasih sudah menyempatkan waktu.

Hari ini kita akan membahas [TOPIC] selama [X] menit.
Tujuan kita adalah [GOAL].

Apakah ada yang ingin ditambahkan ke agenda?"
```

---

### 5.2 Discussion (variable)

**Best Practices:**
- Active listening
- Take notes
- Ask clarifying questions
- Summarize key points
- Manage time

**Questioning Techniques:**

**Open Questions:**
```
"What are your thoughts on...?"
"Can you tell me more about...?"
"How do you feel about...?"
```

**Probing Questions:**
```
"Can you give me an example?"
"What would that look like?"
"Why do you think that is?"
```

**Closing Questions:**
```
"So if I understand correctly, you want... Is that right?"
"Does this solution address your needs?"
"What would be the next step?"
```

---

### 5.3 Action Items (10 minutes)

**At the end of discussion:**

```
"Baik, jadi kesimpulan kita hari ini:

ACTION 1: [Description] - [Owner] - [Deadline]
ACTION 2: [Description] - [Owner] - [Deadline]

Apakah ada yang perlu ditambahkan?"
```

---

### 5.4 Closing (5 minutes)

```
1. Summarize key decisions
2. Confirm next steps
3. Schedule next meeting if needed
4. Thank client
```

**Closing Script:**
```
"Baik, terima kasih untuk waktunya hari ini.

Ringkasan:
- [Key Point 1]
- [Key Point 2]
- [Key Point 3]

Next steps:
- [Action 1]
- [Action 2]

Kita akan schedule next meeting untuk [DATE/TIME/IF NEEDED].

Terima kasih dan sampai jumpa!"
```
