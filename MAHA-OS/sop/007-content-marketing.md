## 4. BLOG POST PROCESS

### Step 1: Research
```
1. Keyword research
2. Competitor analysis
3. Topic ideation
4. Outline creation
```

### Step 2: Write
```
1. Hook (first 100 words)
2. Introduction
3. Main body (H2/H3 structure)
4. Conclusion + CTA
5. Meta description
```

### Step 3: Optimize
```
1. Primary keyword in title
2. Keywords in H2/H3
3. Internal links (2-3)
4. External links (1-2)
5. Images with alt text
6. Meta description
```

### Step 4: Publish
```
1. Upload to CMS
2. Set featured image
3. Add categories/tags
4. Publish
5. Submit to search console
```

## 5. SOCIAL MEDIA PROCESS

### 5.1 Content Creation

```
Platform-Specific Content:

Instagram:
- Image: 1080x1080 or 1080x1350
- Caption: 150-300 words
- Hashtags: 20-30
- CTA: Swipe up/link in bio

Facebook:
- Image: 1200x630
- Post: 50-100 words
- CTA: Comment/Save/Share

LinkedIn:
- Post: 150-300 words
- Image: 1200x627
- CTA: Thought-provoking question

TikTok:
- Video: 15-60 seconds
- Hook: First 3 seconds
- Caption: 50-100 words
- Hashtags: 3-5
```

### 5.2 Hashtag Strategy

## 6. VIDEO CONTENT

### Process
```
1. Script writing (500-1000 words)
2. Recording
3. Editing
4. Thumbnail
5. Upload + SEO
6. Publish + promote
```

### YouTube SEO Checklist
```
☐ Title: Primary keyword + hook
☐ Description: 200-300 words
☐ Tags: 10-15 keywords
☐ Thumbnail: Custom, eye-catching
☐ Chapters: Add timestamps
☐ Cards: Add 2-3 cards
☐ End screen: Subscribe + related video
```
