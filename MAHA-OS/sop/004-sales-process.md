## 4. LEAD QUALIFICATION

### 4.1 Qualification Criteria (BANT)

| Criteria | Questions |
|----------|-----------|
| **Budget** | "Berapa budget yang dialokasikan?" |
| **Authority** | "Siapa yang membuat keputusan?" |
| **Need** | "Apa kebutuhan utamanya?" |
| **Timeline** | "Kapan butuh selesai?" |

### 4.2 Qualification Score

```
Score 8-10: HOT LEAD → Immediate follow-up
Score 5-7:  WARM LEAD → Schedule call
Score 1-4:  COLD LEAD → Nurture campaign
```

## 5. COLD OUTREACH

### 5.1 WhatsApp Script

**Initial Message:**
```
Halo [Nama], salam kenal!

Saya [Nama] dari [Company].

Saya lihat [perusahaan klien] bergerak di [industry]. 
Kami bantu bisnis seperti [contoh] untuk [benefit].

Boleh saya kirimkan proposal untuk [topik]?

Terima kasih!
```

**Follow-up Script:**
```
Halo [Nama], apakah sudah sempat lihat proposal saya?

Kalau ada pertanyaan, saya siap bantu.

Semoga bisa collaborate ya 🙏
```

### 5.2 Email Template

**Subject:** Proposal untuk [Nama Perusahaan] - [Benefit]

**Body:**
```
Halo [Nama],

Saya [Nama] dari [Company].

Saya ingin membantu [Nama Perusahaan] untuk [benefit yang spesifik].

Selama [X] tahun, kami sudah bantu [X] klien untuk [hasil yang dicapai].

Beberapa hasil yang sudah kami capai untuk klien sejenis:
- [Result 1]
- [Result 2]
- [Result 3]

Boleh saya schedule call 15 menit untuk discuss lebih lanjut?

Best regards,
[Nama]
```

## 7. NEGOTIATION

### 7.1 Common Objections

| Objection | Response |
|-----------|----------|
| "Terlalu mahal" | "Bandingkan dengan ROI yang akan Anda dapat..." |
| "Perlu pikir-pikir" | "Apa yang perlu dipertimbangkan?" |
| "Sudah punya vendor" | "Ok, kapan contract berakhir?" |
| "Budget belum tersedia" | "Kapan budget akan ready?" |

### 7.2 Negotiation Tips

```
1. Selalu tanya budget sebelum kasih harga
2. Price anchoring dengan package tertinggi
3. Tawarkan payment terms jika perlu
4. Selalu confirm next steps sebelum tutup call
5. Follow-up dalam 24 jam
```

## 8. CLOSING

### 8.1 Closing Techniques

**Assumptive Close:**
```
"Jadi kita mulai dengan package [X] ya?"
```

**Question Close:**
```
"Apa yang perlu saya jelaskan lagi sebelum kita mulai?"
```

**Urgency Close:**
```
"Jika kita sign hari ini, kita bisa mulai besok..."
```
