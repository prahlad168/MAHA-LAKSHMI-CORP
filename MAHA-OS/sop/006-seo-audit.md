## 1. PURPOSE

```
Standar proses untuk SEO audit website client.
```

## 2. AUDIT SCOPE

```
Technical SEO:
- Site speed
- Mobile friendliness
- Crawlability
- Indexability
- Security

On-Page SEO:
- Title tags
- Meta descriptions
- Heading structure
- Content optimization
- Image optimization

Off-Page SEO:
- Backlink profile
- Social signals
- Brand mentions

Content:
- Quality
- Relevance
- Freshness
- Keyword targeting
```

---

## 3. AUDIT CHECKLIST

### 3.1 Technical SEO

| Check | Tool | Status |
|-------|------|--------|
| Site speed (PageSpeed) | PageSpeed Insights | ☐ |
| Mobile friendliness | Google Mobile Test | ☐ |
| HTTPS enabled | Browser check | ☐ |
| XML sitemap exists | Browser check | ☐ |
| Robots.txt configured | Browser check | ☐ |
| Canonical tags | Browser check | ☐ |
| 404 pages handled | Screaming Frog | ☐ |
| Redirect chains | Screaming Frog | ☐ |
| Broken links | Screaming Frog | ☐ |

### 3.2 On-Page SEO

| Check | Status |
|-------|--------|
| Title tags optimized | ☐ |
| Meta descriptions written | ☐ |
| H1 unique per page | ☐ |
| H2-H6 structure correct | ☐ |
| Internal linking | ☐ |
| Image alt tags | ☐ |
| URL structure clean | ☐ |
| Schema markup | ☐ |

### 3.3 Content

| Check | Status |
|-------|--------|
| Content quality high | ☐ |
| Keywords naturally used | ☐ |
| Content length adequate | ☐ |
| Duplicate content | ☐ |
| Thin content pages | ☐ |
