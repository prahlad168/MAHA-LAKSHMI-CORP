## 3. MORNING BRIEF (9:00 AM)

### 3.1 Checklist

```
□ Check emails and messages
□ Review today's tasks
□ Check calendar for meetings
□ Check pipeline updates
□ Priority setting for today
```

### 3.2 Template

```markdown
# Morning Brief - [DATE]

## Today's Priority
1. [Priority 1]
2. [Priority 2]
3. [Priority 3]

## Scheduled Meetings
- [Time]: [Meeting topic]

## Tasks to Complete
- [ ] Task 1
- [ ] Task 2
- [ ] Task 3

## Blockers from Yesterday
- [Blocker 1]
- [Blocker 2]

## Notes
[Additional notes]
```
