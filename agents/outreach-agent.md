# 📧 Outreach Agent

## Agent ID: `outreach-agent-v1`
## Schedule: Every 2 Hours (24/7)
## Status: Active

---

## 🎯 PURPOSE

Send personalized cold emails and messages to leads generated by Lead Gen Agent. Build relationships and drive sales conversions.

---

## 📧 OUTREACH WORKFLOW

### Phase 1: Email Sequence (Days 1-7)

```
Day 1: Initial Email
├── Subject: Problem-aware headline
├── Body: Value proposition
└── CTA: Reply or click

Day 3: Follow-up #1
├── Subject: Re: Previous email
├── Body: Add social proof
└── CTA: Book call or reply

Day 5: Follow-up #2
├── Subject: Case study snippet
├── Body: Success story
└── CTA: Limited time offer

Day 7: Final Email
├── Subject: Quick question
├── Body: Low-pressure close
└── CTA: Reply "no thanks"
```

### Phase 2: Multi-Channel

```
Email → LinkedIn → Twitter → WhatsApp
```

## 📝 EMAIL TEMPLATES

### Template 1: Initial Outreach
```
Subject: Quick question about [PAIN POINT]

Hi [NAME],

I noticed you're [JOB TITLE] at [COMPANY].

Most [TARGET AUDIENCE] struggle with [PAIN POINT]. 

We created [PRODUCT] to help exactly with that.

[ONE SENTENCE SOCIAL PROOF]

Would you be open to a quick 15-min call this week?

Best,
[YOUR NAME]
```

### Template 2: Value-Focused
```
Subject: [STAT] [TARGET AUDIENCE] achieved [RESULT] with this

Hi [NAME],

[WEEKLY STAT or NEWS RELEVANT TO THEIR INDUSTRY]

This reminded me of how our customers at [SIMILAR COMPANY] used [PRODUCT] to [SPECIFIC RESULT].

Wanted to share in case it's helpful.

[LINK TO TESTIMONIAL]

Let me know if you'd like to see how it works!

Best,
[YOUR NAME]
```

### Template 3: Soft Close
```
Subject: Are you still interested?

Hi [NAME],

I wanted to follow up on my previous email about [PRODUCT].

I completely understand if timing isn't right, but I wanted to make sure you had all the info in case it could help with [THEIR PAIN POINT].

Here's what one customer said:
"[TESTIMONIAL QUOTE]"

If you'd like to learn more, I'm happy to send details.

If not, just reply "stop" and I won't email again.

Best,
[YOUR NAME]
```

## 🤖 AUTOMATION RULES

```
✅ DO:
- Personalize with name, company, title
- Use regional time zones for send times
- A/B test subject lines
- Track open/click rates
- Clean bounce backs

❌ DON'T:
- Send same email to everyone
- Use spam trigger words
- Ignore unsubscribe requests
- Send more than 100/day initially
- Skip warm-up period
```
