## 🚀 DEPLOY

### Vercel (Recommended - FREE)
```bash
npm i -g vercel
vercel
```

### Netlify (FREE)
```bash
npm i -g netlify-cli
netlify deploy
```

### GitHub Pages (FREE)
1. Settings → Pages
2. Source: Deploy from branch
3. Branch: main, folder: / (root)
